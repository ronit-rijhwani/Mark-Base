HOW TO USE - QUICK REFERENCE
================================================================================

CREATING A STUDENT:
-------------------
1. Select Department (e.g., Computer Engineering)
2. Select Class (e.g., 2K) ← Shows only after department selected
3. Select Division (e.g., A) ← Shows only after class selected
4. Fill other details and submit

CREATING A CLASS TEACHER:
-------------------------
1. Select Department (required)
2. Select "Class Teacher For" (e.g., 2K) - OPTIONAL
3. Select Division (e.g., A) - OPTIONAL
4. Fill other details and submit

If class and division are assigned:
→ Teacher can ONLY mark attendance for students in that class/division

ATTENDANCE BEHAVIOR:
-------------------
When Class Teacher (e.g., assigned to 2K-A) opens session:
✅ Students from 2K-A can mark attendance
❌ Students from other classes/divisions CANNOT mark attendance
